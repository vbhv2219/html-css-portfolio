Portfolio Website (HTML & CSS)

This is my first portfolio website created as part of a web development assignment.  
The website is built using **HTML5** and **CSS** and showcases my skills, projects, and contact details.

Semantic HTML5 layout (header, nav, section, footer)
- Responsive design using Flexbox
- About Me section
- Projects showcase
- Contact form
- Clean and simple UI

Technologies Used

- HTML5
- CSS3 (Flexbox & Box Model)

  Project Structure

1. **Home** – Hobby and its benefits  
2. **About** – Short bio  
3. **Projects** – Three sample projects  
4. **Contact** – Contact form  
5. **Footer** – Social media links  

---

How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
